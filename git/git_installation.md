#installation de GIT

### Linux (Debian/Ubuntu/...)
 ```
 sudo dnf install git-all
 ```


### Linux (Debian/Ubuntu/...)
 ```
 sudo apt install git-all
 ```

 ### Windows 

 Cliquer sur le lien [pour télécharger GIT](https://git-scm.com/downloads "Lien pour télécharger Git sur le site officiel").
